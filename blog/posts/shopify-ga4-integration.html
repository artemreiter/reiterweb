Shopify + GA4 Integration: Complete Setup Guide

Shopify's native GA4 integration is broken. It misses 30-40% of transactions and doesn't track the customer journey properly. Here's how to fix it with a proper server-side setup.

The Problem with Shopify's Native Integration

Shopify's built-in Google channel has serious limitations:

  • Doesn't track checkout steps (critical for funnel analysis)
  • Misses refunds and order updates
  • No server-side tracking (blocked by iOS 14+)
  • Limited product data in purchase events

The Solution: Server-Side GTM

For accurate tracking, you need Google Tag Manager Server-Side. Here's the complete setup:

Step 1: Create GTM Server Container

  1. Go to tagmanager.google.com
  2. Create new account ‚Üí Server container
  3. Choose "Manually provision tagging server"
  4. Deploy to Google Cloud (or use Stape for easier setup)

Step 2: Install GTM on Shopify

Add this to your theme.liquid in the <head>:

<!-- Google Tag Manager -->
<script>(function(w,d,s,l,i){w[l]=w[l]||[];w[l].push({'gtm.start':
new Date().getTime(),event:'gtm.js'});var f=d.getElementsByTagName(s)[0],
j=d.createElement(s),dl=l!='dataLayer'?'&l='+l:'';j.async=true;j.src=
'https://www.googletagmanager.com/gtm.js?id='+i+dl;f.parentNode.insertBefore(j,f);
})(window,document,'script','dataLayer','GTM-XXXXXXX');</script>

Step 3: Configure Data Layer

Create a custom pixel or use an app like Elevar to push e-commerce data:

window.dataLayer = window.dataLayer || [];
window.dataLayer.push({
  event: 'purchase',
  ecommerce: {
    transaction_id: '{{ order.order_number }}',
    value: {{ order.total_price | money_without_currency }},
    tax: {{ order.tax_price | money_without_currency }},
    shipping: {{ order.shipping_price | money_without_currency }},
    currency: '{{ order.currency }}',
    items: [{% for line_item in order.line_items %}{
      item_id: '{{ line_item.product_id }}',
      item_name: '{{ line_item.title | escape }}',
      item_category: '{{ line_item.product.type | escape }}',
      price: {{ line_item.price | money_without_currency }},
      quantity: {{ line_item.quantity }}
    }{% unless forloop.last %},{% endunless %}{% endfor %}]
  }
});

Step 4: Test Everything

Before going live:

  • Place a test order
  • Check Real-time reports in GA4
  • Verify transaction_id is unique
  • Confirm revenue matches Shopify admin
